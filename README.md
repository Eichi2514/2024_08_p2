주요 기능 첫번째
2차원 배열을 이용한 좌표 계산입니다

객체끼리의 충돌 판정이나 공격 모션과 객체의 충돌을 확인하려면
각 객체의 위치의 확인 방법이 필요하였고 화면에 보이시는바와 같이
저는 각 객체의 좌표 x축, y축을 2차원 배열의 인덱스로 구하였습니다

![11](https://github.com/user-attachments/assets/6cc34d44-462b-43f8-b089-831615c56b93)

주요 기능 두번째
자바스크립트 키이벤트를 사용한 객체의 이동 및 공격 함수입니다

이동 조작 키가 눌릴때나 눌려있을때 캐릭터가 이동하는거 뿐만 아니라 좌표도 바뀌어야 함으로
css로 보이는 화면에서 이동하도록 하고 ajax(에이잭스)로 좌표도 바뀌도록 하였습니다

공격 조작 키는 연속 공격을 막기위해 키에서 손을 땠을때
즉 키가 올라왔을때만 실행되게 만들어 ajax(에이잭스)로 공격 범위 내의
좌표의 값을 받아와 그 값에 따라 다른 함수들이 실행되게 만들었습니다

![13](https://github.com/user-attachments/assets/1e81afad-30db-46d9-9dbd-b2c6eb0faa16)

웹소켓을 사용한 방식에 대해 간단히 말씀드리자면
웹소켓 연결을 지원하는 클래스를 추가한 후 사용자가 채팅을 전송하면
해당 채팅을 DB에 저장하고 이를 웹소켓을 통해 다른 클라이언트로 전송하도록 구현하였습니다
웹소켓으로 전송된 채팅은 자바스크립트에서 이벤트로 감지하고
받은 데이터를 채팅 형식에 맞게 가공하여 채팅창에 추가하였습니다
이 과정을 통해 실시간 채팅 기능을 구현했습니다
